🧩 Module Specifications
Module 1 — Global Macro & Opening Indicators

Layout: 4 equal cards in a row
Each card: metric name, large price, change value + % badge (green/red pill — reuse existing badge style)
GIFT Nifty card gets an extra "GAP-UP" / "GAP-DOWN" / "FLAT" badge based on sign of changePct
Every card has a ⓘ icon top-right with glassmorphism tooltip:

GIFT Nifty: "Predicts Nifty 50 opening. Premium = Gap-Up. Discount = Gap-Down vs last close."
Brent Crude: "India imports 85% of its oil. Above $90/bbl is Bearish — hits inflation, Paint & Aviation margins."
USD/INR: "Rupee weakening = capital flight (Bearish). But boosts IT & Pharma export earnings (Selective Positive)."
US 10Y Yield: "The gravity of all markets. Rising yields pull FII money from India to safer US returns."



Module 2 — Institutional Flows (Big Money)

Layout: Full-width card with two side-by-side grouped bar charts (FII left, DII right) using Recharts BarChart
Each group: Buy (green), Sell (red), Net (blue or amber based on sign)
Y-axis in ₹ Crore
Below charts: Net Flow Summary row — "FII Net: ▲/▼ ₹X Cr" + "DII Net: ▲/▼ ₹X Cr" + "Combined: ▲/▼ ₹X Cr" color coded
Card ⓘ tooltip: "FIIs are 'Hot Money' driving rallies and crashes. DIIs (mutual funds) act as the market cushion. Combined net flow determines trend strength."

Module 3 — Market Breadth & Sentiment

Layout: Two cards side by side

Card A — Advance/Decline:

Horizontal stacked bar: green (advances) | gray (unchanged) | red (declines), counts labeled inside segments
Below: A/D Ratio (e.g. 1.4:1) + sentiment label: STRONG BREADTH / MIXED / WEAK
ⓘ tooltip: "Internal health check. If Nifty is up but A/D is negative, the rally is narrow and fragile — a warning sign."

Card B — India VIX Gauge:

SVG semi-circle needle gauge, hand-coded

Zones: 0–15 green, 15–20 amber, 20–30 red
Needle animates to VIX value on load (transition: transform 1s ease)
Below: VIX value + label: CALM / CAUTION / FEAR / PANIC


ⓘ tooltip: "The Fear Index. <15: Calm. 15–20: Caution. >20: Fear. >25: Panic. High VIX = expensive options, expect sharp moves."

Module 4 — Sector Rotation Heatmap

Layout: Full-width card
Visual: Treemap via CSS Flexbox (no external lib) — block sizes proportional to weight
Color scale (only new styling decision here):

> +1.5% → #16A34A
0 to +1.5% → #4ADE80 at 60% opacity
-1.5% to 0 → #F87171 at 60% opacity
< -1.5% → #DC2626


Each block: sector name + +X.XX% centered and bold
Hover on block: mini popup with simulated Buy/Sell/Net breakdown
Card ⓘ tooltip: "Sector Rotation reveals where Smart Money is moving today. Consistent green in Defense + Red in IT = risk-off rotation."
